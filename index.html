Virtual Land is now Virtual in your Hand

There is a seperate vland based on the real pincodes of the world. Every Vland graphics are integrated with unity SDK and hosted in the seperate cluster of zones

Space Blocks Processes

The main Technology to make vland work smoother, faster and less-sized is with Unity and WebGL.
Processes :-
low poly blocks should be made by blender and send to unity and it is assembled with automation script (made with Unity project tiny)
exported with WebGL and send to a corresponding located hardware (Server which runs these files) to fetch a link from own secondary hosted zone and updated link is automatically uploaded to zotork app with iframe codes.
For WebGl in general we use babelon.js to run 3d models in web [example: Asset store showcase, web explorations, etc]
